                                           -Readme File-

1. Open the .py file in any python friendly environment, like vs code.
2. Make sure you have installed some libraries like pandas, numpy and matplotlib from the terminal like powershell using pip.
3. Imaginary values will be getting ignored in the graphs. this will give a lots of garbage error and should be ignored likewise.
4. It will show a graph in which- blue curve is for original x signal
                                  orange curve is for the approach first denoise and then deblur.
                                  green curve is for the approach first deblur and then denoise.
5. At last, the code will print 2 values of MSE and the lesser MSE is more efficient.
